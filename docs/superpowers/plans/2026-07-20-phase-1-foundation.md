# HRMS Mobile App — Phase 1: Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Get a Flutter project running on Android with the app's theme (colors/fonts
matching the design) and a working bottom-navigation shell (Home / Attendance /
Leave / Me tabs, each showing a placeholder screen).

**Architecture:** Feature-folder Flutter app (`lib/screens/<feature>/`), `Provider`
for shared state (added now, used from Phase 2 onward), `IndexedStack` +
`BottomNavigationBar` for tab navigation so each tab keeps its own history.

**Tech Stack:** Flutter (stable channel), Dart, `provider` package, `google_fonts`
package (for "Plus Jakarta Sans"), Android emulator for testing.

## Global Constraints

- Target platform for this phase: Android only (per spec — iOS deferred).
- All data is hardcoded/fake — no network calls in this phase.
- Primary accent color: `#10B981` (green), gradient partner `#059669`.
- Background color: `#F1F5F9`. Primary text: `#0f172a`. Secondary text: `#64748b` /
  `#94a3b8`.
- Font: "Plus Jakarta Sans" (fallback Manrope, Figtree) — loaded via `google_fonts`,
  no manual font file downloads needed.
- Card style convention (reused every phase): white background, `border-radius`
  14–22px, soft shadow — defined once in the theme, not repeated ad hoc per screen.
- Verification for this phase is manual/visual (run the app, compare to the
  mockup) per the spec's testing approach — no automated widget tests required
  yet.

### Task 1: Install Flutter SDK and verify the toolchain

**Files:** none (environment setup only)

- [ ] **Step 1: Download the Flutter SDK**

  Go to https://docs.flutter.dev/get-started/install/windows/mobile and download
  the Windows Flutter SDK zip. Extract it to `C:\src\flutter` (avoid `Program Files`
  — spaces in the path cause tooling issues). Do not put it inside `C:\Projects`.

- [ ] **Step 2: Add Flutter to PATH**

  Open Windows Settings → search "environment variables" → "Edit the system
  environment variables" → "Environment Variables" → under "User variables" select
  `Path` → "Edit" → "New" → add `C:\src\flutter\bin`. Click OK on all dialogs, then
  close and reopen any terminal.

- [ ] **Step 3: Verify Flutter is on PATH**

  Run: `flutter --version`
  Expected: prints a Flutter version number (e.g. `Flutter 3.x.x`) with no
  "command not found" error.

- [ ] **Step 4: Install Android Studio**

  Download and install Android Studio from https://developer.android.com/studio.
  During setup, let it install the "Android SDK", "Android SDK Platform-Tools", and
  "Android Virtual Device" components (default options do this).

- [ ] **Step 5: Create an Android emulator (virtual phone)**

  In Android Studio: More Actions → Virtual Device Manager → "Create Device" →
  pick a phone (e.g. Pixel 7) → pick a system image (e.g. latest API level shown,
  download it if prompted) → Finish.

- [ ] **Step 6: Run flutter doctor and resolve issues**

  Run: `flutter doctor`
  Expected: a checklist. Any `[✗]` line must be fixed before continuing — the
  output tells you exactly what command to run (commonly
  `flutter doctor --android-licenses`, then accept all license prompts by typing
  `y`). Re-run `flutter doctor` until Flutter and Android toolchain both show
  `[✓]`.

- [ ] **Step 7: Install the VS Code Flutter extension**

  In VS Code: Extensions panel → search "Flutter" → install the official
  "Flutter" extension (it pulls in the Dart extension automatically). This gives
  you a "Run" button and hot-reload while editing.

### Task 4: Build the app theme

**Files:**
- Create: `hrms_app/lib/theme/app_theme.dart`

**Interfaces:**
- Produces: `AppTheme.colors` (a class with named color constants) and
  `AppTheme.themeData` (a `ThemeData` object), both importable as
  `import 'package:hrms_app/theme/app_theme.dart';` from any screen file.

- [ ] **Step 1: Write the theme file**

  Create `hrms_app/lib/theme/app_theme.dart`:
  ```dart
  import 'package:flutter/material.dart';
  import 'package:google_fonts/google_fonts.dart';

  /// Colors and text styles pulled directly from the HRMS design mockup.
  class AppColors {
    static const primary = Color(0xFF10B981);
    static const primaryDark = Color(0xFF059669);
    static const background = Color(0xFFF1F5F9);
    static const cardBackground = Color(0xFFFFFFFF);
    static const textPrimary = Color(0xFF0F172A);
    static const textSecondary = Color(0xFF64748B);
    static const textMuted = Color(0xFF94A3B8);
    static const danger = Color(0xFFEF4444);
    static const warning = Color(0xFFF59E0B);
    static const border = Color(0xFFE2E8F0);
  }

  class AppTheme {
    static ThemeData get themeData {
      final baseTextTheme = GoogleFonts.plusJakartaSansTextTheme();
      return ThemeData(
        scaffoldBackgroundColor: AppColors.background,
        primaryColor: AppColors.primary,
        colorScheme: ColorScheme.fromSeed(
          seedColor: AppColors.primary,
          background: AppColors.background,
        ),
        textTheme: baseTextTheme.apply(
          bodyColor: AppColors.textPrimary,
          displayColor: AppColors.textPrimary,
        ),
        cardTheme: CardTheme(
          color: AppColors.cardBackground,
          elevation: 2,
          shadowColor: Colors.black.withOpacity(0.06),
          shape: RoundedRectangleBorder(
            borderRadius: BorderRadius.circular(16),
          ),
        ),
        bottomNavigationBarTheme: const BottomNavigationBarThemeData(
          selectedItemColor: AppColors.primary,
          unselectedItemColor: AppColors.textMuted,
          backgroundColor: AppColors.cardBackground,
          type: BottomNavigationBarType.fixed,
        ),
      );
    }
  }
  ```

### Task 6: Build the bottom-navigation shell and wire up the app

**Files:**
- Create: `hrms_app/lib/app.dart`
- Modify: `hrms_app/lib/main.dart`

**Interfaces:**
- Consumes: `HomeScreen`, `AttendanceScreen`, `LeaveScreen`, `ProfileScreen`
  (from Task 5); `AppTheme.themeData` (from Task 4).
- Produces: `HrmsApp` — the root widget, used by `main.dart`.

- [ ] **Step 1: Write the navigation shell**

  Create `hrms_app/lib/app.dart`:
  ```dart
  import 'package:flutter/material.dart';
  import 'theme/app_theme.dart';
  import 'screens/home/home_screen.dart';
  import 'screens/attendance/attendance_screen.dart';
  import 'screens/leave/leave_screen.dart';
  import 'screens/profile/profile_screen.dart';

  class HrmsApp extends StatelessWidget {
    const HrmsApp({super.key});

    @override
    Widget build(BuildContext context) {
      return MaterialApp(
        title: 'HRMS',
        debugShowCheckedModeBanner: false,
        theme: AppTheme.themeData,
        home: const MainTabShell(),
      );
    }
  }

  class MainTabShell extends StatefulWidget {
    const MainTabShell({super.key});

    @override
    State<MainTabShell> createState() => _MainTabShellState();
  }

  class _MainTabShellState extends State<MainTabShell> {
    int _selectedIndex = 0;

    static const _screens = [
      HomeScreen(),
      AttendanceScreen(),
      LeaveScreen(),
      ProfileScreen(),
    ];

    void _onTabTapped(int index) {
      setState(() => _selectedIndex = index);
    }

    @override
    Widget build(BuildContext context) {
      return Scaffold(
        body: SafeArea(
          child: IndexedStack(
            index: _selectedIndex,
            children: _screens,
          ),
        ),
        bottomNavigationBar: BottomNavigationBar(
          currentIndex: _selectedIndex,
          onTap: _onTabTapped,
          items: const [
            BottomNavigationBarItem(icon: Icon(Icons.home_outlined), label: 'Home'),
            BottomNavigationBarItem(icon: Icon(Icons.access_time_outlined), label: 'Attendance'),
            BottomNavigationBarItem(icon: Icon(Icons.event_note_outlined), label: 'Leave'),
            BottomNavigationBarItem(icon: Icon(Icons.person_outline), label: 'Me'),
          ],
        ),
      );
    }
  }
  ```
